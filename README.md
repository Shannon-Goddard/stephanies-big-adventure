# 🚲 Stephanie's Big Adventure

A comic book-styled interactive web experience inspired by Pee-Wee's Big Adventure. Built as a surprise gift.

## What Is This?

A two-page animated experience that leads to a real-world scavenger hunt via QR code and GPS coordinates. No spoilers — you'll have to play to find out.

## Features

- Layered comic book visuals with CSS animations
- Interactive Pee-Wee bike launch with sound effect
- Responsive design (desktop, mobile landscape & portrait)
- Mission briefing page with offline map instructions
- Google Maps integration with precise coordinates

## Tech

Pure HTML, CSS, and vanilla JS. No frameworks, no dependencies. Google Fonts (Bangers) for the comic book typography.
